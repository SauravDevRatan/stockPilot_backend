# StockPilot Backend

A **Node.js & Express** backend for **StockPilot**, built using the **MVC (Model-View-Controller) architecture**. Provides RESTful APIs for user authentication, stock trading data, and integration with the frontend dashboard and landing page. Supports image uploads via **Cloudinary** and secure JWT-based authentication.

## Features
- **User Authentication:** Registration, login, and JWT-based token verification
- **API Endpoints:** Serve stock data, holdings, and user account info to the dashboard
- **Cloudinary Integration:** Upload and manage user profile images
- **MVC Architecture:** Structured for maintainability and scalability
- **Secure:** Input validation and proper error handling for API requests

---

## Tech Stack
- **Backend:** Node.js, Express
- **Database:** MongoDB (or whichever DB you used)
- **Authentication:** JWT (JSON Web Tokens)
- **File Storage:** Cloudinary for profile images
- **Deployment:** Render
